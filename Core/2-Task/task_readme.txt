
Task 文件夹中主要存放能够独立的完成某种特定的任务的程序:

    其重点是需要完成特定的任务，所以必须代码片段特定用于解决某个问题
    不是FUNC文件中具有泛化性能的函数，故TASK文件夹是封装的最上层，无法被再次封装。
    针对不同的系统，往往需要重写。
    通常需要多个FUNC函数结合完成，设计多个硬件，没有init函数。
    
该文件夹可存放的任务举例：
    助行器中，采集IMU数据并根据IMU传感器协议 每隔 10ms 读取出 角速度/加速度/角度      注：func中的IMU读取不涉及每隔多久采集一次，以后后续IMU数据处理
    助行器中，根据步态相位计算力矩并输出给驱动器的任务
