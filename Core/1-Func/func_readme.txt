
Func 文件夹中

1. 存放比较复杂的硬件设备的子功能实现程序。

譬如CAN设备，发送一帧数据就是子功能，与任务设备不同，往往一个函数就能搞定

2. 存放某种硬件的操作调用接口

3. 存放针对该硬件的，能够独立运行的调试函数。以 TEST 定义

4. 针对力矩传感器，func中的函数仅仅是用于获取数据，不会有后续的处理，譬如数据滤波，这样才能保证func中的函数在不同的任务中不需要修改。


注意：该文件夹中的函数并不针对某种特定的任务，因此可以被封装成其他的函数，

	譬如某个报警任务，需要灯泡闪烁以警示。此时可以新建一个报警函数封装LED的闪烁函数，调用报警函数，即可闪烁灯泡。
